# Codex Learning Log

Codexを動かして学んだことを紹介する静的サイトです。

## 開き方

`index.html` をブラウザで開くだけで表示できます。ビルドや依存パッケージのインストールは不要です。

## GitHub Pages

公開URL:

https://bostonlegal01.github.io/codex01/

Pagesの公開元は `main` ブランチの `/ (root)` を想定しています。

## 構成

- `index.html`: ページ構造と本文
- `styles.css`: レイアウトとビジュアルデザイン
- `script.js`: 気づきタブとチェック項目の操作
- `assets/codex-learning-hero.png`: 生成したヒーロー画像
- `.nojekyll`: GitHub Pagesで静的ファイルをそのまま配信するための設定
